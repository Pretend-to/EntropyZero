# 指令面板 UI 原型

这是一个参考 macOS Spotlight 设计的指令面板 HTML 原型，展示了毛玻璃效果和场景切换功能。

## 功能特性

### 🎨 视觉设计
- **毛玻璃效果**: 使用 `backdrop-filter: blur()` 实现真实的毛玻璃背景
- **半透明背景**: 渐变背景遮罩，营造沉浸式体验
- **流畅动画**: 200ms 缓动动画，提供自然的交互反馈
- **响应式布局**: 适配不同屏幕尺寸，最小边距保护

### ⌨️ 交互方式
- **Cmd+K**: 唤起/关闭指令面板
- **Escape**: 关闭面板
- **方向键**: 在建议列表中导航
- **Enter**: 选择当前项
- **背景点击**: 关闭面板

### 🔄 场景切换
- **默认场景**: 显示常用功能入口
- **AI 场景** (`/ai`): 聊天界面，支持对话交互
- **搜索场景** (`/search`): 全局搜索，实时结果高亮
- **模板场景** (`/template`): 模板选择器，支持分类筛选
- **工作流场景** (`/workflow`): 工作流执行器，显示可用的自动化流程
- **设置场景** (`/settings`): 应用设置面板，支持各种配置选项
- **帮助场景** (`/help`): 快捷键和使用指南

### 🎯 智能特性
- **前缀检测**: 自动识别 `/` 开头的场景切换指令
- **实时搜索**: 输入即搜索，结果实时更新
- **关键词高亮**: 搜索结果中的匹配关键词高亮显示
- **模板分类**: 支持按类别筛选模板（项目管理、软件开发、设计、营销等）
- **工作流状态**: 显示工作流的当前状态和可执行性
- **设置同步**: 实时切换设置选项，支持开关控件
- **快捷键指南**: 内置帮助系统，显示所有可用的快捷键和场景切换命令
- **上下文感知**: 根据当前场景调整 UI 布局

## 使用方法

1. 在浏览器中打开 `index.html`
2. 按 `Cmd+K` 唤起指令面板
3. 尝试以下操作：
   - 输入 `/ai` 进入 AI 助手模式
   - 输入 `/search` 进入搜索模式
   - 输入 `/template` 查看模板库
   - 输入 `/workflow` 查看工作流
   - 输入 `/settings` 打开设置
   - 输入 `/help` 查看帮助
   - 输入普通文字进行搜索
   - 使用方向键导航选项

## 技术实现

### CSS 特性
- `backdrop-filter`: 毛玻璃模糊效果
- `transition`: 流畅的动画过渡
- `rgba()`: 半透明颜色
- `box-shadow`: 深度阴影效果

### JavaScript 功能
- 事件监听和处理
- 场景路由管理
- 键盘导航逻辑
- 动态内容渲染

### 浏览器兼容性
- **Chrome/Edge**: 完全支持
- **Firefox**: 需要启用 `layout.css.backdrop-filter.enabled`
- **Safari**: 需要 `-webkit-backdrop-filter` 前缀

## 下一步开发

这个原型为后续的 React/TypeScript 实现提供了设计参考：

1. **组件化**: 将各个场景拆分为独立的 React 组件
2. **状态管理**: 使用 Zustand 或 Redux 管理全局状态
3. **API 集成**: 连接真实的 AI 服务和搜索接口
4. **测试覆盖**: 添加单元测试和属性测试

## 设计原则

- **简洁性**: 界面简洁，操作直观
- **一致性**: 遵循 macOS 设计语言
- **可访问性**: 支持键盘导航和屏幕阅读器
- **性能**: 流畅的动画和快速响应
- **扩展性**: 易于添加新的场景和功能
